# 🕵️ Beijing Agent — 大客户销售背调助手

大客户销售拜访前的公司背调 Agent。输入目标公司名称，自动完成公司画像、经营分析、行业定位，生成结构化的会客策略报告。

## 两种模式

| 模式 | 命令 | 内容 | 耗时 |
|------|------|------|------|
| 快速 | `/beijing 公司名` | 公司画像、经营分析、竞争定位、会客策略 | ~5 分钟 |
| 深度 | `/beijing --deep 公司名` | 快速模式 + 组织架构、关键人物地图、决策人画像、需求推断 | ~20 分钟 |

## 使用方式

在 Claude Code 中加载 `SKILL.md` 作为 skill：

1. 将 `SKILL.md` 放入 `~/.claude/skills/beijing/` 目录
2. 在对话中输入 `/beijing 目标公司名` 即可触发

## 报告示例

快速背调 → `~/Desktop/{公司名}_背调报告_快速_{日期}.md`
深度背调 → `~/Desktop/{公司名}_背调报告_深度_{日期}.md`

### 示例：帆软深度背调

见 [examples/帆软_背调报告_深度_20260512.md](examples/帆软_背调报告_深度_20260512.md)

## 核心原则

1. **销售视角翻译** — 每个事实都回答"对销售有什么用？"
2. **并行搜索** — 最多 12 个搜索同时发起，节省时间
3. **信息可信度标记** — 🔴官方 🟡第三方 🟠推测 ⚪待核实
